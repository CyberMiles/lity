Getting Started
===============

Download the Prebuilt lityc
---------------------------

See `Lity releases <https://github.com/CyberMiles/lity/releases>`_
for the latest release.

Dependencies
````````````

- Ubuntu 16.04: :code:`apt install libz3-dev`
